The Icon component is a versatile building block for creating consistent and reusable icons in your application. It supports custom sizes, colors, and click handlers, making it easy to integrate icons into various parts of your UI.

## Basic Icon

This example demonstrates how to use the Icon component with a simple SVG path.

```jsx
<Icon>
  <path d="M12 17.27L18.18 21l-1.64-7.03L22 9.24l-7.19-.61L12 2 9.19 8.63 2 9.24l5.46 4.73L5.82 21z" />
</Icon>
```
## Sized Icons
You can easily control the size of the Icon component using the `size` prop. This example shows icons of different sizes.
```jsx
<div style={{ display: 'flex', gap: 'var(--spacing-large)', alignItems: 'center' }}>
  <Icon size={16}>
    <path d="M12 17.27L18.18 21l-1.64-7.03L22 9.24l-7.19-.61L12 2 9.19 8.63 2 9.24l5.46 4.73L5.82 21z" />
  </Icon>
  <Icon size={32}>
    <path d="M12 17.27L18.18 21l-1.64-7.03L22 9.24l-7.19-.61L12 2 9.19 8.63 2 9.24l5.46 4.73L5.82 21z" />
  </Icon>
  <Icon size={48}>
    <path d="M12 17.27L18.18 21l-1.64-7.03L22 9.24l-7.19-.61L12 2 9.19 8.63 2 9.24l5.46 4.73L5.82 21z" />
  </Icon>
</div>
```
## Clickable Icon
The Icon component supports the `onClick` event, allowing you to add interactivity to your icons.
```jsx
<Icon onClick={() => alert('Icon clicked!')}>
  <path d="M12 17.27L18.18 21l-1.64-7.03L22 9.24l-7.19-.61L12 2 9.19 8.63 2 9.24l5.46 4.73L5.82 21z" />
</Icon>

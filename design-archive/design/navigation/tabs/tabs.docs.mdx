The `Tabs` component allows users to navigate between different views within the same context. It's useful for organizing content and providing a streamlined user experience.

## Component Usage

The basic usage of the `Tabs` component involves passing an array of `TabItem` objects to the `tabs` prop. Each `TabItem` should have a `label` (the text displayed on the tab) and a `component` (the content to be rendered when the tab is active).

```jsx
const tabs = [
  { label: 'Tab 1', component: <div>Content 1</div> },
  { label: 'Tab 2', component: <div>Content 2</div> },
  { label: 'Tab 3', component: <div>Content 3</div> },
];
<Tabs tabs={tabs} />
```
## Simple Content Tabs
This example demonstrates how to use the `Tabs` component with simple text content in each tab.
```jsx
const simpleTabs = [
  { label: 'Tab One', component: <p>This is the content for the first tab.</p> },
  { label: 'Tab Two', component: <p>This is the content for the second tab.</p> },
  { label: 'Tab Three', component: <p>This is the content for the third tab.</p> }
];
<Tabs tabs={simpleTabs} />
```
## Preselected Tab
You can specify a default active tab using the `defaultActiveTab` prop. This prop accepts the index of the tab that should be active when the component initially renders.
```jsx
const sampleTabs = [
  { label: 'Directions', component: <div>Directions Content</div> },
  { label: 'Sailing', component: <div>Sailing Content</div> },
  { label: 'Enterprise', component: <div>Enterprise Content</div> },
];
<Tabs tabs={sampleTabs} defaultActiveTab={1} />
```

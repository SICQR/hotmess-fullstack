The `Logo` component displays the company logo and name, with options for a slogan, custom image, and different sizes.
It can be rendered as a link or a static element.

## Default Logo

Displays the default Acme logo.

```jsx
<Logo />
```

## Custom Logo with Slogan and Link
This example shows how to use a custom name, slogan, and link.
```jsx
<Logo name="Innovate Inc." slogan="Building the Future" href="/about" />
```
